How to install on TradingView

1. Open TradingView → any chart → click Pine Editor at bottom of screen.
2. Click 📋 Copy to clipboard on a script above → paste into the editor (replace any default code).
3. Click Save → name it (e.g. "OF Proxy") → click Add to chart.
4. Repeat for each script. OF Proxy goes in lower pane (separate). Footprint Lite + VA Profile overlay candles.

Wire webhook alerts to your backend

1. Need a public URL pointing at this Flask server. For local dev: ngrok http 5001 → copy the https://xxx.ngrok.io URL.
2. In TV: right-click chart → Add alert → Condition: OF Proxy → Any alert() function call.
3. In Notifications tab, check Webhook URL: https://YOUR-NGROK.ngrok.io/api/order-flow/tv-alert
4. Message field: leave as {{ '{{message}}' }} (Pine already builds the JSON).
5. Save alert. When OF Proxy fires R3/R6/R7 → TV POSTs JSON → Flask persists to outputs/order_flow/tv_alerts.jsonl + main alerts pipeline → appears in /order-flow dashboard.
Note: Webhook alerts require TradingView Pro+ plan ($14.95/mo). Free plan only gets popup/email/push.
